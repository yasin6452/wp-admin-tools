# wp-admin-tools

مجموعه ابزارهای سریع مدیریت ادمین وردپرس

این ریپازیتوری شامل سه اسکریپت ساده و کاربردی PHP برای مدیریت کاربران ادمین در سایت‌های وردپرسی است.

---

## فایل‌های موجود

### 1. `create-admin.php`
ایجاد کاربر ادمین جدید با اطلاعات دلخواه.

### 2. `check-user.php`
بررسی وجود کاربر و نمایش آیدی آن.

### 3. `reset-password.php`
ریست کردن پسورد کاربر موجود.

---

## نحوه استفاده

1. فایل مورد نظر را در **روت** public html سایت وردپرسی خود (همان پوشه‌ای که `wp-load.php` داخل آن است) آپلود کنید.
2. در مرورگر آدرس فایل را بزنید، مثلاً:
   - `https://your-site.com/create-admin.php`
3. بعد از اجرا، **فایل را فوراً حذف کنید** (برای امنیت).

> ⚠️ **هشدار امنیتی**: این اسکریپت‌ها فقط برای مواقع اضطراری (وقتی دسترسی به داشبورد ندارید) استفاده شوند و بعد از استفاده حذف شوند.

---
